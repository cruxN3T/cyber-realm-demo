# Cyber Realm — Water Sector Edition · Playable Demo

The official free browser demo of **Cyber Realm: Water Sector Edition**, the cybersecurity
trading card game by The Cat in the Black Hat.

**Play it:** https://play.cyberrealmtcg.com (or this repo's GitHub Pages URL)

You play the defender — a small water facility's blue team — against a Script Kiddie attacker
walking the cyber kill chain. Defend the plant for 20 turns using the NIST Cybersecurity
Framework: Identify, Protect, Detect, Respond, Recover.

- 97 real cards from the printed set, on the official play mat
- Strict rulebook rules: realm-icon costs, allocation economy, operator combat, hand limit 7
- Built-in 🎓 Coach with threat reads, suggested plays, and security lessons
- Educational callouts + end-of-game Incident Debrief
- Single self-contained HTML file — no server, no tracking, no signup

Get the tabletop game: https://cyberrealmtcg.com
